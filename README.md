# Legal Precedent Recommendation & Case Outcome Prediction

An AI-powered legal analytics system that recommends relevant past cases and predicts case outcomes using data mining and machine learning techniques.

---

## 🚀 Features

- 🔍 **Case Recommendation System**
  - Uses TF-IDF + Cosine Similarity to suggest similar legal cases

- 📊 **Outcome Prediction**
  - Predicts case outcome (civil/criminal) using Naive Bayes

- 🔗 **Frequent Pattern Mining**
  - Applies Apriori algorithm to find commonly co-occurring legal sections

- 🧠 **Case Clustering**
  - Uses DBSCAN to group similar legal cases

- 🌐 **Web Scraping**
  - Extracts real case data from Indian Kanoon

---

## 🛠️ Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- MLxtend
- BeautifulSoup (Web Scraping)
